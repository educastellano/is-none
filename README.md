# is-none

Checks wether a value is none (null or undefined).

# Install

    npm install is-none --save

# Usage


```js
    import isNone from 'is-none'
    
    let x
    let y = null
    let z = 0

    isNone(x) // true
    isNone(y) // true
    isNone(z) // false
```
